# The Wealth Paradox: Global Suicide Rates & Economic Well-being

A data science analysis of global suicide rates against macroeconomic indicators (GDP, inflation, employment) and the UNDP Human Development Index, spanning 180+ countries from 1990 to 2022.

## Live Site

[https://hafsahnasir.github.io/suicide-rates-economic-analysis](https://hafsahnasir.github.io/suicide-rates-economic-analysis)

## Data Sources

- **WHO** (via Kaggle) — Suicide rates 1990–2022
- **World Bank** — GDP, inflation, employment indicators
- **UNDP HDI** — Human Development Index

## Tech Stack

Python, pandas, NumPy, statsmodels, scipy, matplotlib, seaborn

## Project Structure

```
data/       — Raw CSV datasets
```
